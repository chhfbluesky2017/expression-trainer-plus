# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-07-18

### Added
- 新增三种工作模式切换功能：训练模式、润色模式、语音输入模式
- 训练模式：保持原有功能，包含实时反馈和词库分析
- 润色模式：语音转录后自动通过大模型进行润色处理
- 语音输入模式：重新设计为浮动小球，可悬浮于桌面任意位置
- 小球支持鼠标拖拽移动，位置自动保存
- 小球右键呼出扇形菜单，可切换到训练/润色模式
- 语音转文字直接输出到系统当前活动窗口的文本输入框
- 支持 Alt+S 全局快捷键控制语音输入开关
- 小球录音状态有明确的视觉反馈（红色脉冲动画 + 波形动画）
- 语音输入模式实时显示中间识别结果，说话过程中文字逐字显示
- Ollama 自动启动功能：启动程序时若当前配置为 Ollama 且未运行，后台异步自动启动
- 设置页面新增 Ollama 状态检测和手动启动按钮，实时显示运行状态
- 主界面左下角新增 Ollama 状态显示栏，实时显示运行状态，未运行时显示启动按钮
- 支持 electron-packager 打包，生成可执行文件

### Changed
- 语音输入模式从全屏界面改为独立浮动窗口（frameless transparent always-on-top）
- 语音转文字结果不再显示在软件内置文本框，而是直接输出到系统焦点所在的文本输入区域
- 统一模式切换入口，顶部栏显示三种模式按钮
- 语音输入模式新增实时文字气泡，说话过程中即可看到中间识别结果
- 语音输入模式支持实时录入：说话过程中中间识别结果通过退格替换方式实时输出到系统窗口

### Fixed
- 修复语音模式下无法切换回其他模式的问题
- 修复语音输入重复文字问题：sherpa-onnx 在 reset 后 isEndpoint 可能仍为 true，导致同一句 final 文本被反复输出
- 修复通过应用启动 Ollama 时找不到已下载模型的问题（使用 spawn 并传递完整环境变量）
- 修复 Ollama 状态检测误判问题（增加 JSON 响应验证）

## [1.5.0] - 2026-07-15

### Fixed
- 修复可能导致系统不稳定的问题：ASR 引擎资源正确释放、音频处理增加防抖、优化缓冲区管理

## [1.0.0] - 2026-07-15

### Added
- 新增浅色主题支持，默认使用浅色主题
- 新增主题切换按钮（顶栏 🌙/☀️ 图标）
- 主题设置持久化保存到 localStorage
- 窗口尺寸和位置记忆功能，重启后自动恢复
- Ollama 本地模型自动刷新功能，下拉显示实际可用模型列表

### Changed
- 重构 CSS 使用 CSS 变量定义主题配色
- 设置页面和训练规则页面适配主题切换
- 词库 Playground 页面适配主题切换
- 训练规则页面返回按钮移至右上角
- 移除设置窗口和训练规则窗口的顶部菜单栏
- 修复终端中文乱码问题（添加 chcp 65001 和 console 编码转换）
- 顶部菜单栏从英文改为中文
- 「开始录制」按钮改为「开始」
- 优化语音录制启动速度

### Fixed
- 修复主题切换时的 FOUC（闪烁）问题
- 修复训练规则页面返回按钮与标题重叠问题
- 修复语音录制无法录入问题（asrReady 状态不同步）
